# C++ 基础与进阶学习仓库
学习范围：C++语法、编码规范、STL、底层原理、并发编程

## 📚 参考书籍
1. 《C++ Primer 第5版》
2. 《Effective C++ 第三版》
3. 《More Effective C++》
4. 《Effective STL》
5. 《STL源码剖析》（侯捷）
6. 《深度探索C++对象模型》
7. 《C++并发编程实战》

## 📖 阅读顺序
1. 通读 **《C++ Primer》**，夯实全部语法与基础 STL
2. 研读 **《Effective C++》** → **《More Effective C++》**，规范编码风格
3. 学习 **《Effective STL》**，熟练用好标准库
4. 研读 **《STL源码剖析》**，理解 STL 底层实现
5. 学习 **《深度探索C++对象模型》**，吃透 C++ 底层机制
6. 最后学习 **《C++并发编程实战》**，掌握并发、多线程核心
