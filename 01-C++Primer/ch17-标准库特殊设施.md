# 第 17 章 标准库特殊设施

日常开发不高频、但关键场景必备的小众标准库工具，不属于基础语法。

## 17.1 tuple 元组

- 存放**不同类型**的一组数据，长度固定，比 `pair` 支持更多成员
- `get<索引>(tuple对象)` 获取元素，`tuple_size` 获取元素个数
- 场景：函数一次性返回多个不同类型结果、简易异构数据打包

## 17.2 bitset 位集

- 固定长度二进制位容器，专门做位运算（与、或、异或、置 1、清 0、翻转）
- 按下标访问某一位、统计 1 的个数、整体转为整数/字符串
- 场景：权限掩码、状态标记、高效布尔数组

## 17.3 正则表达式 regex

C++11 正则全套工具：

- `regex`：正则表达式对象；`smatch`：字符串匹配结果
- `regex_match`：完整匹配；`regex_search`：查找子串；`regex_replace`：替换
- 支持迭代遍历所有匹配项，语法兼容 ECMAScript 正则规则

## 17.4 随机数

摒弃 `rand()`，现代 C++ 随机数架构：

1. **随机引擎**（如 `mt19937` 梅森旋转）：生成底层无符号随机原始数
2. **分布器**（均匀整数/浮点、正态、伯努利等）：映射到目标数值区间

优势：可控种子、分布均匀、可复现，适合仿真、抽奖、测试。

## 17.5 计时工具（chrono）

| 组件 | 说明 |
|------|------|
| 时钟 | `system_clock`（墙上时钟）、`steady_clock`（单调计时）、`high_resolution_clock` |
| 时长 | `duration`：时分秒毫秒纳秒单位转换 |
| 时间点 | `time_point` |

常用：统计代码耗时、定时、时间戳转换。

## 17.6 可选：特殊数值类型

智能指针相关补充、`numeric_limits` 等工具。

## 学习优先级

**业务开发必学**：随机数、`chrono` 计时、正则、`tuple` 日常经常用。
