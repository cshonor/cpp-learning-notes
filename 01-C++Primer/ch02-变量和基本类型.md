# 第 2 章 变量和基本类型

数据类型是程序的基础。本章讲述 C++ 的基本内置类型、复合类型、常量限定符、类型推断机制，并初步介绍如何自定义数据结构。

## 2.1 基本内置类型

- C++ 基础数据类型：字符、整型、浮点数等
- 字面值常量：十进制、八进制、十六进制
- 转义序列

## 2.2 变量

- 变量的定义与初始化
- 未初始化变量的危险性
- 声明与定义的关系（支持分离式编译）
- 标识符命名规范
- 变量作用域嵌套规则

## 2.3 复合类型

- **引用（Reference）**：对象别名
- **指针（Pointer）**：间接访问机制
- 复杂类型声明：指向指针的指针、指向指针的引用等

## 2.4 const 限定符

- `const` 定义不可变变量，多文件间共享
- 对常量的引用、常量指针
- **顶层 const**：对象本身是常量
- **底层 const**：指向的对象是常量
- C++11：`constexpr` 变量与常量表达式

## 2.5 处理类型

- `typedef` 与 C++11 `using`：类型别名
- `auto`：编译器自动推断变量类型
- `decltype`：根据表达式推断要定义的变量类型

## 2.6 自定义数据结构

- 使用 `struct` 组合基本类型（`Sales_data`）
- C++11 类内初始值
- 规范编写头文件（`#ifndef` 等保护符）

## 小结

本章涵盖 C++ 底层数据表示与类型管理的基础机制，是后续学习类与高级特性的先决条件。
