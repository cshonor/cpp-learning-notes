# 第 4 章 表达式

本章介绍 C++ 丰富的运算符，以及它们作用于内置类型时的操作与隐式类型转换规则。对于含多个运算符的复杂表达式，需理解**优先级（precedence）、结合律（associativity）和求值顺序（order of evaluation）**。

## 4.1 基础

- 运算符基本概念：一元、二元运算符
- 优先级、结合律、求值顺序对表达式结果的影响
- 未指定求值顺序时可能引发的错误

## 4.2–4.10 各种运算符详解

| 类别 | 内容 |
|------|------|
| 算术运算符 | `+`、`-`、`*`、`/`、`%` |
| 逻辑和关系运算符 | `&&`、`||`、`<`、`==` 等；逻辑与/或的**短路求值** |
| 赋值运算符 | `=`、`+=` 等复合形式 |
| 递增和递减 | `++`、`--`；前置与后置区别；建议优先使用前置版本 |
| 成员访问 | 点运算符 `.`、箭头运算符 `->` |
| 条件运算符 | `?:` 及嵌套用法 |
| 位运算符 | `~`、`<<`、`>>`、`&`、`^`、`|` |
| sizeof | 返回表达式或类型所占字节数 |
| 逗号运算符 | `,` |

## 4.11 类型转换

- **隐式类型转换**：运算对象类型不一致时编译器自动转换（算术转换、数组转指针、指针转换、转为布尔类型等）
- **显式类型转换**：`static_cast`、`const_cast` 等
