# 第 11 章 关联容器

关联容器与顺序容器不同，元素通过**关键字（key）**而非位置来存储和访问。

## 核心类型

| 类别 | 容器 |
|------|------|
| 有序关联容器 | `map`、`set`、`multimap`、`multiset`（基于红黑树/平衡树） |
| 无序关联容器（C++11） | `unordered_map`、`unordered_set` 等（基于哈希函数） |

## pair 类型

- 辅助关联容器（如 `map`）存储关键字-值对
- `pair` 类型及其基本操作

## 特有操作

- 对关键字类型的限制
- `insert`、`emplace` 插入元素（检查返回值）
- `erase` 删除元素
- 下标操作与查找函数：`find`、`count`、`lower_bound`、`equal_range`
