# 第 1 章 开始

本章介绍 C++ 的大部分基础内容，包括类型、变量、表达式、语句及函数，帮助读者具备编写、编译及运行简单程序的能力。

## 1.1 编写一个简单的 C++ 程序

- C++ 程序的基本结构
- `main` 函数四要素：返回类型、函数名、形参列表、函数体
- 如何编译和运行程序

## 1.2 初识输入输出

- C++ 标准库 `iostream` 机制
- `std::cin`：标准输入
- `std::cout`：标准输出
- `std::cerr`：标准错误
- `endl` 操纵符的基本用法

## 1.3 注释简介

- 单行注释：`//`
- 界定符对注释：`/* */`
- 注意：注释界定符不能嵌套

## 1.4 控制流

- `while` 循环
- `for` 循环
- `if` 条件语句
- 利用循环读取数量不定的输入数据

## 1.5 类简介

- 初步引入 C++ 核心特性——类（class）
- 不涉及从头实现类，通过假定的 `Sales_item` 类学习：
  - 定义类对象
  - 调用成员函数（方法）
  - 类对象之间的基本运算（相加、赋值）

## 1.6 书店程序

本章总结性实战：结合输入输出、控制流与 `Sales_item` 类，编写书店程序——读取多条销售记录，汇总输出各本书总销售额和平均售价。
