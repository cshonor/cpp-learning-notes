# 第 10 章 泛型算法

标准库未在每个容器上定义所有功能，而是提供一组独立于特定容器的**泛型算法**，用于对序列（通常是一段迭代器范围）进行操作。

## 基础算法与重排

- 查找、计数、填充、拷贝：`find`、`accumulate` 等
- 排序和重排：`sort`、`unique` 等
- **泛型算法本身永远不会改变底层容器的大小**

## 定制操作与 Lambda 表达式

- 默认运算符（`<`、`==`）不适用时，向算法传递自定义谓词函数
- C++11 **Lambda 表达式**：捕获列表、参数绑定（`bind`）机制

## 再探迭代器

| 类型 | 用途 |
|------|------|
| 插入迭代器 | 自动执行插入操作以扩展容器 |
| 流迭代器 | 绑定到 IO 流 |
| 反向迭代器 | 逆序遍历 |
| 移动迭代器 | 移动语义遍历 |

## 特定容器算法

- `list`、`forward_list` 因内存结构特殊，提供专用成员函数算法
- 链表专用：`sort`、`merge`、`splice` 等，比通用算法性能更好
