# 第 5 章 语句

本章介绍 C++ 的控制流（flow-of-control）语句，支持程序跳出简单顺序执行，实现条件分支、循环和跳转等复杂执行路径。

## 5.1–5.2 简单语句与作用域

- 表达式语句、空语句、复合语句（块）
- 控制结构（`if`、`for` 等）内部定义变量的**块作用域**

## 5.3 条件语句

### if 语句

- 根据条件决定执行路径
- `if-else` 嵌套
- **悬垂 else（dangling else）**问题及花括号的正确使用

### switch 语句

- 在若干固定选项（整数类型常量）中做出选择
- `case` 标签、`break` 的阻断作用、`default` 标签
- 在 `switch` 内部定义变量的规则

## 5.4 迭代语句（循环）

| 语句 | 特点 |
|------|------|
| **while** | 条件为真时重复执行；适合迭代次数未知的场景 |
| **传统 for** | 初始化、条件、表达式三部分；适合按步骤迭代 |
| **范围 for（C++11）** | 简便遍历容器、数组或其他序列（`vector`、`string` 等） |
| **do while** | 先执行循环体再检查条件；保证至少执行一次 |

## 5.5 跳转语句

- **break**：终止最近循环或 `switch`
- **continue**：终止当前迭代，开始下一次迭代
- **goto**：无条件跳转（书中建议尽量避免）

## 5.6 try 语句块和异常处理

- **throw 表达式**：引发异常
- **try 语句块**：包裹可能出错的代码
- **catch 子句**：捕获和处理异常
- `<stdexcept>` 头文件中的标准异常类（如 `runtime_error`）
