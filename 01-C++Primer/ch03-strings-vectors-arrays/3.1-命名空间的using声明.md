# 3.1 命名空间的 using 声明

## 本节讲什么

本节讲解「命名空间的 using 声明」。

## 要点

- 通过 `using` 声明简化代码（如 `using std::cin;`）
- 使用标准库名字时无需每次都加 `std::` 前缀

## 示例

```cpp
#include <iostream>
#include <string>
using std::cin;
using std::cout;
using std::endl;
using std::string;
int main() {
    string line;
    getline(cin, line);
    cout << line << endl;
    return 0;
}
```
