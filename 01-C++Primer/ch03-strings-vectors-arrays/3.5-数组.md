# 3.5 数组

## 本节讲什么

介绍 C++ 的**内置数组**类型。数组在存放元素上与 `vector` 类似，但**大小固定**，灵活性较差。深入讨论数组的定义、初始化、下标访问，以及**指针与数组的紧密联系**（数组名在很多情况下会自动转换为指向首元素的指针），并提示开发者在现代 C++ 中应尽量使用标准库类型（`vector`、`string`）而非低级的内置数组和指针。

## 要点

- **定义**：`int arr[10];`、`int arr[] = {1,2,3};`（大小由初值推断）
- **大小固定**：创建后不能改变元素个数
- **下标访问**：`arr[i]`，下标从 0 开始；越界是未定义行为
- **数组与指针**：数组名在多数表达式中转为指向首元素的指针；`auto p = arr;` 等价于 `auto p = &arr[0];`
- **`begin(arr)`/`end(arr)`**（C++11）：获取数组首、尾后指针，便于算法与循环
- **优先使用 `vector`/`string`**：自动管理大小与内存，更安全

## 示例

```cpp
#include <iostream>
#include <iterator>

int main() {
    int arr[3] = {1, 2, 3};

    // 下标访问
    arr[0] = 10;
    std::cout << arr[0] << std::endl;

    // 数组名 → 指向首元素的指针
    int *p = arr;
    std::cout << *p << " " << *(p + 1) << std::endl;

    // 用 begin/end 遍历
    for (int *beg = std::begin(arr), *end = std::end(arr);
         beg != end; ++beg)
        std::cout << *beg << " ";
    std::cout << std::endl;

    return 0;
}
```
