# 小结

## 本节讲什么

本章是 **C++ 标准库学习的起点**。回顾 `string`、`vector`、迭代器与内置数组的关系，并强调：开发者应**优先选用**标准库提供的 `string` 和 `vector`，只在必要时才考虑底层的数组或指针。

## 要点

- **`using` 声明**：简化 `std::` 前缀，头文件中慎用
- **`string`**：可变长字符串，配合 `cctype` 处理字符
- **`vector`**：可变长容器，`push_back` 添加，下标只访问已有元素
- **迭代器**：统一的遍历接口，`begin`/`end`、`cbegin`/`cend`
- **内置数组**：大小固定，与指针关系密切；现代 C++ 中作为备选
- **多维数组**：本质是数组的数组；可用范围 for 与类型别名简化

## 示例

```cpp
#include <iostream>
#include <string>
#include <vector>

int main() {
    std::vector<std::string> words{"C++", "Primer", "Chapter3"};

    for (const auto &w : words)
        std::cout << w << " ";
    std::cout << std::endl;

    // 标准库类型组合：比裸数组 + 指针更安全、更易用
    return 0;
}
```
