# 小结

## 本节讲什么

本章是 C++ 标准库学习的起点。应**优先选用**标准库的 `string` 和 `vector`，只在必要时才考虑底层数组或指针。

## 示例

```cpp
#include <iostream>
#include <string>
#include <vector>
int main() {
    std::vector<std::string> words{"C++", "Primer"};
    for (auto &w : words)
        std::cout << w << " ";
    std::cout << std::endl;
    return 0;
}
```
