# 3.4 迭代器介绍

## 本节讲什么

本节讲解「迭代器介绍」。

## 要点

- 迭代器：更通用的访问 `string` 字符和 `vector` 元素的方法
- `begin` / `end`、`cbegin` / `cend` 获取迭代器
- 解引用、递增遍历容器
- 迭代器的算术运算

## 示例

```cpp
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v{1, 2, 3};
    for (auto it = v.cbegin(); it != v.cend(); ++it)
        std::cout << *it << " ";
    std::cout << std::endl;
    return 0;
}
```
