# 3.4 迭代器介绍

## 本节讲什么

引入**迭代器（iterator）**机制——一种更通用的访问 `string` 字符和 `vector` 元素的方法。讲解如何使用 **`begin`/`end`**（以及 **`cbegin`/`cend`**）获取迭代器，如何通过解引用和递增遍历容器，并介绍迭代器的算术运算。

## 要点

- 迭代器类似指针，指向容器中的元素；`*it` 解引用，`++it` 移到下一元素
- **`begin()`**：首元素迭代器；**`end()`**：尾后迭代器（不指向元素，表示结束）
- **`cbegin()`/`cend()`**：只读迭代器，不能通过其修改元素
- 遍历模式：`for (auto it = v.begin(); it != v.end(); ++it) { ... *it ... }`
- **迭代器算术**：两个迭代器相减得距离；`it + n` 前进 n 个元素（仅 vector/string 等支持）
- 若容器为空，`begin() == end()`

## 示例

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <cctype>

int main() {
    std::vector<int> v{1, 2, 3, 4, 5};

    // 用迭代器遍历
    for (auto it = v.cbegin(); it != v.cend(); ++it)
        std::cout << *it << " ";
    std::cout << std::endl;

    // 迭代器算术
    auto mid = v.begin() + v.size() / 2;
    std::cout << "middle: " << *mid << std::endl;

    // string 同样支持迭代器
    std::string s = "hello";
    for (auto it = s.begin(); it != s.end(); ++it)
        *it = static_cast<char>(std::toupper(*it));
    std::cout << s << std::endl;

    return 0;
}
```
