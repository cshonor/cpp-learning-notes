# 3.3 标准库类型 vector

## 本节讲什么

本节讲解「标准库类型 vector」。

## 要点

- C++ 最常用的容器类型，存放给定类型对象的可变长序列
- 定义、列表初始化
- `push_back`：运行时动态添加元素
- 下标运算符的正确用法（**不能用下标添加新元素**）

## 示例

```cpp
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v{1, 2, 3, 4, 5};
    for (int x : v)
        std::cout << x << " ";
    std::cout << std::endl;
    return 0;
}
```
