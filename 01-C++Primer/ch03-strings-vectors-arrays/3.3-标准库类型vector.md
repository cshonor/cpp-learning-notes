# 3.3 标准库类型 vector

## 本节讲什么

介绍 C++ 中最常用的容器类型 **`vector`**，用于存放某种给定类型对象的可变长序列。讲解 `vector` 的定义、列表初始化、如何使用 **`push_back`** 在运行时向其中动态添加元素，以及正确使用下标运算符的方法（**特别强调：不能用下标去添加新元素**）。

## 要点

- `#include <vector>`；`vector<T>` 存放类型 `T` 的对象序列
- **定义**：`vector<int> v;`、`vector<int> v(10);`（10 个默认值）、`vector<int> v{1,2,3};`（列表初始化）
- **`push_back`**：在末尾追加元素，是运行时增容的正确方式
- **下标 `v[i]`**：仅用于访问**已存在**的元素；`v[v.size()]` 是越界
- **不能用下标添加**：`v[10] = 42;` 在 `size() <= 10` 时是错误的
- 可用 `size()`、`empty()` 查询状态

## 示例

```cpp
#include <iostream>
#include <vector>

int main() {
    std::vector<int> v;              // 空 vector
    for (int i = 1; i <= 5; ++i)
        v.push_back(i);              // 正确：动态添加元素

    std::vector<int> v2{1, 2, 3};    // 列表初始化

    for (std::size_t i = 0; i != v.size(); ++i)
        std::cout << v[i] << " ";    // 下标访问已有元素
    std::cout << std::endl;

    // 错误示例（不要这样做）：
    // v[10] = 100;  // 越界！不能用下标添加新元素

    return 0;
}
```
