# 3.2 标准库类型 string

## 本节讲什么

本节讲解「标准库类型 string」。

## 要点

- 可变长字符序列 `string` 类
- 对象的定义与初始化
- 读写操作、拼接、比较等基本操作
- `cctype` 头文件：`isalpha`、`toupper` 等，单独处理字符串中的特定字符

## 示例

```cpp
#include <iostream>
#include <string>
int main() {
    std::string s1 = "hello", s2;
    s2 = s1;
    s1 = "world";
    std::cout << s1 << " " << s2 << std::endl;  // world hello
    return 0;
}
```
