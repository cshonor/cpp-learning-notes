# 3.2 标准库类型 string

## 本节讲什么

详细讲解表示可变长字符序列的 **`string` 类**。涵盖 `string` 对象的定义与初始化、读写操作、对象上的基本操作（如拼接、比较），以及如何借助 **`cctype`** 头文件中的标准库函数（如 `isalpha`、`toupper` 等）单独处理字符串中的特定字符。

## 要点

- `#include <string>`；`string` 可变长，自动管理内存
- **定义与初始化**：`string s;`、`string s = "hello";`、`string s(10, 'c');` 等
- **读写**：`cin >> s` 读单词；`getline(cin, s)` 读整行（含空格）
- **操作**：`+` 拼接、`==`/`!=`/`>` 比较、`size()`/`empty()` 等
- **`cctype`**：`isalpha`、`isdigit`、`isspace`、`tolower`、`toupper` 等，参数为 `int`（通常传 `unsigned char` 转换后的字符）

## 示例

```cpp
#include <iostream>
#include <string>
#include <cctype>

int main() {
    std::string s1 = "hello", s2;
    s2 = s1;
    s1 = "world";
    std::cout << s1 << " " << s2 << std::endl;  // world hello

    std::string greeting = s1 + ", " + s2;     // 拼接
    std::cout << greeting << std::endl;

    // 使用 cctype 处理单个字符
    for (unsigned char c : s1) {
        if (std::isalpha(c))
            std::cout << static_cast<char>(std::toupper(c));
    }
    std::cout << std::endl;

    return 0;
}
```
