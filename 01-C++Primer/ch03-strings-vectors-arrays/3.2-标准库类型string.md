# 3.2 标准库类型 string

## 本节讲什么

`std::string` 是 C++ 处理文本的**核心类**，也是后面写 HFT 策略、处理**订单 / 行情 / 日志**文本的基础。本节重点：**初始化方式**、**读写操作**、**内存管理逻辑**，以及和 **C 风格字符串**的本质区别——`string` **自动管理内存**，不必手动 `malloc` / `free`，能避开大量泄漏坑。

## 要点

- **`#include <string>`**；可变长，**RAII 自动释放**内存
- **初始化**：默认空串、拷贝、字面值、`string(n, c)` 重复字符
- **读写**：`cin >> s` 读单词；`getline(cin, s)` 读整行；`cout << s` 输出
- **操作**：`+` / `+=` 拼接、`==` 比较、`size()` / `empty()` / `[]` / `substr`
- **vs C 字符串**：`char*` + 手动分配 vs `string` 随内容增长自动扩缩
- **HFT**：FIX 报文解析、日志拼接、合约代码 / 订单号处理（见文末拓展）

## 一、定义与初始化

```cpp
#include <string>

std::string s1;                    // 空串 ""
std::string s2 = "hello";          // 字面值
std::string s3(s2);                // 拷贝
std::string s4(10, 'c');           // "cccccccccc"
std::string s5 = std::string(5, 'x');  // "xxxxx"
```

| 写法 | 结果 |
|------|------|
| `string s;` | 默认构造，空串 |
| `string s = "abc";` | 从 C 风格字面量拷贝内容 |
| `string s(other);` | 深拷贝 `other` 的字符序列 |
| `string s(n, c);` | `n` 个字符 `c` |

列表初始化（C++11）：

```cpp
std::string s{"IF2406"};   // 与 = "IF2406" 等价
```

## 二、读写操作

### 输出

```cpp
std::string msg = "order sent";
std::cout << msg << std::endl;
```

### 输入：`>>` vs `getline`

| 方式 | 行为 | 典型场景 |
|------|------|----------|
| `cin >> s` | 跳过前导空白，读到**下一个空白**为止 | 读单词、合约代码、单个 token |
| `getline(cin, s)` | 读到**换行**（含中间空格） | 读整行日志、FIX 字段行 |

```cpp
std::string word, line;
std::cin >> word;              // "IF2406" 从 "IF2406 100" 里读出
std::getline(std::cin, line);  // 读剩余或下一整行
```

读 FIX / 管道分隔文本时，常配合 `find('|')`、`substr` 按字段切，而不是一次 `>>` 读完。

### 拼接与比较

```cpp
std::string a = "buy", b = "IF2406";
std::string c = a + " " + b;   // "buy IF2406"
a += " done";

if (a == b) { /* ... */ }
if (a.size() > 0) { /* 或 !a.empty() */ }
```

## 三、内存管理：string vs C 风格字符串

| | **`std::string`** | **C 风格 `char*` / `char[]`** |
|---|-------------------|-------------------------------|
| 长度 | **可变**，随内容自动增长/收缩 | 固定缓冲或裸指针，易越界 |
| 内存 | 构造时分配、析构时**自动释放**（RAII） | `malloc` / `new[]` 须配对 `free` / `delete[]` |
| 拼接 | `+` / `+=` 自动重新分配 | `strcat` 等须保证目标缓冲足够大 |
| 安全性 | 边界由 `size()` 管理；`at(i)` 越界检查 | 无长度信息，**缓冲区溢出**常见 |

```cpp
// ❌ C 风格：易忘释放或越界
char* p = new char[100];
// ... 若异常路径未 delete[] → 泄漏

// ✅ string：作用域结束自动析构
{
    std::string fix = "8=FIX.4.2|35=D|";
}   // 内存自动回收
```

**与 C 接口互操作**（读旧 API、写 C 函数时）：

```cpp
std::string s = "hello";
const char* c = s.c_str();   // 只读 C 风格视图；s 改内容后可能失效
// s[0] = 'H';               // 再调 c_str() 前勿假设 c 仍有效
```

HFT 新代码优先 `string`；底层网关若只给 `const char*`，可立刻包一层 `std::string msg(ptr);` 再处理。

## 四、`cctype`：按字符处理

`#include <cctype>`，对**单个字符**分类 / 转换（参数类型为 `int`，通常传 `unsigned char` 避免负值 UB）：

| 函数 | 含义 |
|------|------|
| `isalpha(c)` | 字母 |
| `isdigit(c)` | 数字 |
| `isspace(c)` | 空白 |
| `isalnum(c)` | 字母或数字 |
| `toupper(c)` / `tolower(c)` | 大小写转换 |

```cpp
for (unsigned char c : symbol) {
    if (std::isdigit(c))
        /* 数字段 */;
}
```

## 五、HFT 场景小拓展

### 1. FIX 报文（简化示意）

FIX 是交易所常见的**键值对 + 分隔符**文本（`|` 或 `\001` SOH）。`string` 适合整段持有报文，再 `find` / `substr` 取字段：

```cpp
#include <string>

// 简化：用 '|' 代替 SOH
std::string fix = "8=FIX.4.2|35=D|11=OID1001|55=IF2406|38=10|";

std::string tag_11;   // ClOrdID
auto pos = fix.find("11=");
if (pos != std::string::npos) {
    auto start = pos + 3;
    auto end = fix.find('|', start);
    tag_11 = fix.substr(start, end - start);   // "OID1001"
}
```

生产环境会用专业 FIX 库；这里说明 **`string` 是报文容器 + 字段切分的基础**。

### 2. 日志拼接

```cpp
std::string log;
log += "[";
log += std::to_string(order_id);   // C++11
log += "] send ";
log += symbol;
log += " qty=";
log += std::to_string(qty);
// 或一次拼接："[123] send IF2406 qty=10"
```

热路径上若频繁拼接，可考虑 `reserve()` 预分配减少重分配，或固定缓冲 + 长度（进阶优化）；入门阶段 **`string` + `+=` 清晰安全**即可。

### 3. 合约 / 订单文本

- 合约代码：`"IF2406"`、`"600519.SH"` —— `string` 比较、哈希、作 map 键
- 与 [2.6 Sales_data](../ch02-variables-and-basic-types/2.6.1-Sales_data入门与聚合.md) 的 `bookNo` 同类：**业务文本字段用 `string`，数值用算术类型**

## 示例

```cpp
#include <iostream>
#include <string>
#include <cctype>

int main() {
    std::string s1 = "hello", s2;
    s2 = s1;
    s1 = "world";
    std::cout << s1 << " " << s2 << std::endl;  // world hello

    std::string greeting = s1 + ", " + s2;
    std::cout << greeting << std::endl;

    std::string line;
    std::getline(std::cin, line);

    for (unsigned char c : s1) {
        if (std::isalpha(c))
            std::cout << static_cast<char>(std::toupper(c));
    }
    std::cout << std::endl;

    return 0;
}
```
