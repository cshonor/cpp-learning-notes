# 第 14 章 操作重载与类型转换

本章介绍如何让自定义类类型在直观上像内置类型一样使用，核心是通过重载机制给内置运算符赋予新含义。

## 运算符重载

| 类别 | 运算符 |
|------|--------|
| 输入/输出 | `>>`、`<<` |
| 算术和关系 | `+`、`==` 等 |
| 赋值 | 复合赋值、`initializer_list` 赋值 |
| 下标 | `[]` |
| 递增/递减 | `++`、`--` |
| 成员访问 | `*`、`->` |

## 函数调用运算符与函数对象

- 重载 `operator()`
- 重载该运算符的类对象称为**函数对象**，可像函数一样调用
- C++11 **Lambda 表达式**本质：编译器生成的未命名函数对象类的未命名对象

## 类型转换运算符

- 定义特殊成员函数（如 `operator int() const;`），实现从自定义类向其他类型的隐式转换
- 隐式转换与重载运算符混用时的二义性问题及避免方法
