# 2.6 自定义数据结构

## 本节讲什么

作为类的初步体验，带领读者使用 `struct` 关键字将相关的数据元素组织成自定义数据结构（如 `Sales_data`）。介绍 C++11 的**类内初始值**特性，并讲解如何规范编写头文件，以及如何使用头文件保护符（`#ifndef`、`#define`、`#endif`）防止重复包含引起的重新定义错误。

## 要点

- **`struct`**：将多个数据成员组合为用户自定义类型
- **类内初始值**（C++11）：成员声明时可给默认值，如 `unsigned units_sold = 0;`
- **头文件**：声明放在 `.h`/`.hpp`，实现放在 `.cpp`，支持分离式编译
- **头文件保护**：
  ```cpp
  #ifndef SALES_DATA_H
  #define SALES_DATA_H
  // ...
  #endif
  ```
- 现代 C++ 也可用 `#pragma once`（非标准但广泛支持）

## 示例

**Sales_data.h**

```cpp
#ifndef SALES_DATA_H
#define SALES_DATA_H

#include <string>

struct Sales_data {
    std::string bookNo;
    unsigned units_sold = 0;   // 类内初始值
    double revenue = 0.0;
};

#endif
```

**main.cpp**

```cpp
#include <iostream>
#include "Sales_data.h"

int main() {
    Sales_data data1;                                    // 使用类内默认值
    Sales_data data2{"978-0-201-78345-0", 25, 22.5};    // 聚合初始化
    std::cout << data2.bookNo << ": sold " << data2.units_sold << std::endl;
    return 0;
}
```
