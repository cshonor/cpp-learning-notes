# 2.6 自定义数据结构

## 本节讲什么

本节讲解「自定义数据结构」。

## 要点

- 使用 `struct` 组合基本类型（`Sales_data`）
- C++11 类内初始值
- 规范编写头文件（`#ifndef` 等保护符）

## 示例

```cpp
#include <iostream>
#include <string>
struct Sales_data {
    std::string bookNo;
    unsigned units_sold = 0;
    double revenue = 0.0;
};
int main() {
    Sales_data data1, data2{"978-0-201-78345-0", 25, 22.5};
    std::cout << data2.bookNo << std::endl;
    return 0;
}
```
