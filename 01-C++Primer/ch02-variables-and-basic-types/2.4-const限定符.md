# 2.4 const 限定符

## 本节讲什么

介绍如何使用 `const` 定义值不能被改变的变量。深入探讨**对常量的引用**、**常量指针**，并明确区分**顶层 const**（指针/对象本身是常量）和**底层 const**（所指向的对象是常量）。最后引入 C++11 新特性 **`constexpr`** 变量和常量表达式。

## 要点

- **`const int ci = 0;`**：`ci` 的值不可修改
- **常量引用** `const int &r = ci;`：不能通过 `r` 修改所绑定的对象；可绑定到字面量
- **指向常量的指针** `const int *p` / `int const *p`：底层 const，不能通过 `p` 改所指对象
- **常量指针** `int *const p`：顶层 const，`p` 的地址不可变，但可通过 `p` 改对象
- **顶层 const**：对象本身 const；**底层 const**：所指对象 const；可同时存在
- **`constexpr`**：编译期常量，值必须在编译时确定

## 示例

```cpp
#include <iostream>
int main() {
    const int ci = 42;
    const int &cr = ci;       // 对常量的引用

    const int *p = &ci;       // 指向常量的指针（底层 const）
    // *p = 0;                // 错误

    int j = 0;
    int *const cp = &j;       // 常量指针（顶层 const）
    *cp = 100;                // 可以
    // cp = &ci;              // 错误：cp 不能改指向

    constexpr int size = 64;  // 编译期常量
    char buf[size];           // 大小在编译时已知

    return 0;
}
```
