# 2.1 基本内置类型

## 本节讲什么

介绍 C++ 定义的**算术类型**（包括字符、整型数、布尔值和浮点数）以及**空类型**（`void`）。详细讲解**字面值常量**（literal）：十进制、八进制、十六进制的整型字面值，以及字符和字符串字面值等。

## 要点

- **算术类型**：`char`、`int`、`long`、`bool`、`float`、`double` 等，不同机器上大小可能不同
- **void**：不对应具体对象，常用于函数无返回值或通用指针
- **整型字面值**：十进制 `42`；八进制 `042`（前缀 `0`）；十六进制 `0x2A`（前缀 `0x`/`0X`）
- **浮点字面值**：如 `3.14`、`3.14e0`；默认 `double`，后缀 `f`/`F` 表示 `float`
- **字符字面值**：`'a'`；**字符串字面值**：`"hello"`（实际含末尾 `\0`）
- **转义序列**：`\n`、`\t`、`\\`、`\"` 等
- **类型大小因平台而异**：C++ 标准只规定各类型的**最小取值范围**，**没有强制字节数**；不同硬件架构、编译器下同一类型可能占不同字节

## 为什么同一类型在不同机器上大小不一样？

根本原因是**硬件架构**和**编译器/ABI 规则**不同。标准保证「至少能表示多大范围」，具体占几字节由实现决定。

| 类型 | 常见情况 | 注意 |
|------|----------|------|
| **`int`** | 32 位系统通常 4 字节 | 16 位旧系统可能是 2 字节 |
| **`long`** | Windows 32/64 位通常都是 **4 字节** | Linux 64 位上通常是 **8 字节** |
| **`long long`** | 一般 8 字节 | 跨平台相对更统一 |
| **`bool`** | 通常 1 字节 | 个别嵌入式平台可能自定义 |
| **`float` / `double`** | 通常 4 / 8 字节 | 相对固定，嵌入式也可能有差异 |
| **`char`** | 通常 1 字节 | — |

**跨平台开发时**：不要假设 `int`、`long` 的字节数；需要固定宽度时用 `<cstdint>` 里的 `int32_t`、`int64_t` 等（后续章节会用到）。

## 示例

```cpp
#include <iostream>
int main() {
    int dec = 42;
    int oct = 052;       // 八进制，值为 42
    int hex = 0x2A;      // 十六进制，值为 42

    double pi = 3.14;
    bool flag = true;
    char ch = 'A';
    const char *msg = "hello";

    std::cout << dec << " " << oct << " " << hex << std::endl;
    std::cout << pi << " " << flag << " " << ch << " " << msg << std::endl;

    // 查看本机各类型占多少字节（结果因平台而异）
    std::cout << "sizeof(int)=" << sizeof(int)
              << " sizeof(long)=" << sizeof(long)
              << " sizeof(double)=" << sizeof(double) << std::endl;

    return 0;
}
```
