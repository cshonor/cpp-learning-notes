# 2.1 基本内置类型

## 本节讲什么

介绍 C++ 定义的**算术类型**（包括字符、整型数、布尔值和浮点数）以及**空类型**（`void`）。详细讲解**字面值常量**（literal）：十进制、八进制、十六进制的整型字面值，以及字符和字符串字面值等。

## 要点

- **算术类型**：`char`、`int`、`long`、`bool`、`float`、`double` 等，不同机器上大小可能不同
- **void**：不对应具体对象，常用于函数无返回值或通用指针
- **整型字面值**：十进制 `42`；八进制 `042`（前缀 `0`）；十六进制 `0x2A`（前缀 `0x`/`0X`）
- **浮点字面值**：如 `3.14`、`3.14e0`；默认 `double`，后缀 `f`/`F` 表示 `float`
- **字符字面值**：`'a'`；**字符串字面值**：`"hello"`（实际含末尾 `\0`）
- **转义序列**：`\n`、`\t`、`\\`、`\"` 等

## 示例

```cpp
#include <iostream>
int main() {
    int dec = 42;
    int oct = 052;       // 八进制，值为 42
    int hex = 0x2A;      // 十六进制，值为 42

    double pi = 3.14;
    bool flag = true;
    char ch = 'A';
    const char *msg = "hello";

    std::cout << dec << " " << oct << " " << hex << std::endl;
    std::cout << pi << " " << flag << " " << ch << " " << msg << std::endl;
    return 0;
}
```
