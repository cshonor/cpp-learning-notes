# 2.1 基本内置类型

## 本节讲什么

本节讲解「基本内置类型」。

## 要点

- C++ 基础数据类型：字符、整型、浮点数等
- 字面值常量：十进制、八进制、十六进制
- 转义序列

## 示例

```cpp
#include <iostream>
int main() {
    int i = 42;
    double pi = 3.14;
    bool flag = true;
    char ch = 'A';
    std::cout << i << " " << pi << " " << flag << " " << ch << std::endl;
    return 0;
}
```
