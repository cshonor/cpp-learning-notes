# 2.3 复合类型

## 本节讲什么

本节讲解「复合类型」。

## 要点

- **引用（Reference）**：对象别名
- **指针（Pointer）**：间接访问机制
- 复杂类型声明：指向指针的指针、指向指针的引用等

## 示例

```cpp
#include <iostream>
int main() {
    int ival = 42;
    int *p = &ival;       // p 指向 ival
    std::cout << *p << std::endl;  // 解引用
  *p = 0;
    std::cout << ival << std::endl;  // 0
    return 0;
}
```
