# 2.2 变量

## 本节讲什么

本节讲解「变量」。

## 要点

- 变量的定义与初始化
- 未初始化变量的危险性
- 声明与定义的关系（支持分离式编译）
- 标识符命名规范
- 变量作用域嵌套规则

## 示例

```cpp
#include <iostream>
int main() {
    int ival = 1024;
    int &refVal = ival;  // refVal 是 ival 的别名
    refVal = 2048;
    std::cout << ival << std::endl;  // 2048
    return 0;
}
```
