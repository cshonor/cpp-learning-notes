# 1.2 初识输入输出

## 本节讲什么

本节讲解「初识输入输出」。

## 要点

- C++ 标准库 `iostream` 机制
- `std::cin`：标准输入
- `std::cout`：标准输出
- `std::cerr`：标准错误
- `endl` 操纵符的基本用法

## 示例

```cpp
#include <iostream>
int main() {
    std::cout << "Enter two numbers: ";
    int v1 = 0, v2 = 0;
    std::cin >> v1 >> v2;
    std::cout << v1 << " + " << v2 << " = " << v1 + v2 << std::endl;
    return 0;
}
```
