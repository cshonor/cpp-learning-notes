# 1.5 类简介

## 本节讲什么

初步引入 C++ 最重要特性之一的**类（class）**：通过定义类来自定义数据结构。本节通过书中预设好的 `Sales_item` 类，教导读者**无需关心内部实现**，只需了解如何定义类类型的对象、如何利用点运算符 `.` 调用成员函数（如提取书号的 `isbn()`），以及如何使用运算符完成对象间的读取、打印和相加操作。

## 要点

- 类是 C++ 的核心机制，用于定义用户自定义类型
- 定义类对象：`Sales_item item;`
- 点运算符 `.` 调用成员函数：`item.isbn()`
- `Sales_item` 支持 `>>` 读取、`<<` 打印、`+` 相加、`=` 赋值
- 本节只**使用**类，类的定义与实现留待后续章节

## 通俗理解

把 **class** 想成一个定制化的 **「数据盒子 + 工具包」**：

| `Sales_item`（书） | 以后 HFT 里的 `Order`（单） |
|--------------------|----------------------------|
| 盒子里：书号、销量、收入 | 盒子里：委托价、数量、订单号 |
| 工具：`isbn()` 取书号 | 工具：`send()` 发单、`cancel()` 撤单 |
| `>>` 读一条销售记录 | 从网关读入一笔委托 |
| `+` 合并两笔销售 | 合并/汇总订单 |

**不用自己拆盒子找数据**——定义好对象后，用 `.` 调方法就行，例如 `item.isbn()`，比零散传一堆参数清晰。

`Sales_item` 是入门的「友好型例子」：你先学会**怎么用**盒子，后面章节再学**怎么造**盒子。

## 头文件与实现文件

**不是**类必须全写在头文件里。常见做法是分开写：

| 文件 | 放什么 | 作用 |
|------|--------|------|
| **`Sales_item.h`** | 类的**声明**（有哪些方法、长什么样） | 给别人看「怎么调用」 |
| **`Sales_item.cpp`** | 方法的**具体实现**（函数体里怎么写） | 藏实现细节，减少重复编译 |

HFT 项目里常把核心策略实现藏在 `.cpp` 或静态库/动态库里，**对外只暴露头文件接口**——既保护代码，又不影响别人调用。

## 链接：编译器怎么找到实现？

```
main.cpp  #include "Sales_item.h"
    ↓ 编译：编译器从头文件知道 isbn() 存在 → 能通过编译
    ↓
Sales_item.cpp  → 单独编译成目标文件
    ↓ 链接：把 item.isbn() 的调用点和 .cpp 里的实现「绑」在一起
    ↓
可执行文件 → 运行时正确执行 isbn() 里的代码
```

你在 `main` 里写 `item.isbn()` 时：
1. **编译阶段**：只看 `.h`，确认 `isbn()` 名字、参数对得上就行
2. **链接阶段**：去 `.cpp` 编译出的模块里找到 `isbn()` 的真正代码并接上

所以：**头文件给编译器看，实现文件给链接器接。**

## 头文件长什么样？（`Sales_item.h` 极简版）

```cpp
#pragma once              // 防止头文件被重复包含（也可用 #ifndef 守卫）

#include <string>

class Sales_item {
public:
    std::string isbn() const;   // 声明：有这个方法，具体实现在 .cpp
    // ... 其他 public 接口（>>、<<、+ 等）

private:
    std::string bookNo;         // 数据成员：外部不能直接改
    unsigned units_sold = 0;
    double revenue = 0.0;
};
```

别人 `#include "Sales_item.h"` 之后就知道怎么调 `isbn()`，但**看不到**函数体，也**碰不到** `private` 里的 `bookNo`。

**`Sales_item.cpp`（实现放这里）：**

```cpp
#include "Sales_item.h"

std::string Sales_item::isbn() const {
    return bookNo;              // 具体怎么取书号，写在这里
}
```

## 示例

```cpp
// 需配合书中 Sales_item 头文件使用；此处为用法示意
#include <iostream>
#include "Sales_item.h"

int main() {
    Sales_item item;
    std::cin >> item;                          // 从输入读取一条销售记录
    std::cout << item << std::endl;            // 打印该记录
    std::cout << item.isbn() << std::endl;     // 调用成员函数获取书号

    Sales_item item2;
    std::cin >> item2;
    Sales_item sum = item + item2;             // 两笔销售记录相加
    std::cout << sum << std::endl;
    return 0;
}
```
