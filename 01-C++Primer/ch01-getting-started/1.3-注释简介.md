# 1.3 注释简介

## 本节讲什么

讲解 C++ 中的两种注释方式：**单行注释**（以 `//` 开始）和**界定符对形式的注释**（以 `/*` 开始，以 `*/` 结束）。书中特别警告：界定符对形式的注释**不能嵌套**使用，否则会导致编译器产生令人迷惑的错误。

## 要点

- 单行注释：`//` 从该行注释符到行尾均为注释
- 界定符对注释：`/*` … `*/` 之间的内容为注释，可跨行
- **禁止嵌套** `/* */` 注释（如在一个 `/*` … `*/` 内再写 `/*`）
- 注释不影响程序行为，用于说明代码意图

## 示例

```cpp
// 单行注释：说明下面代码的作用
#include <iostream>
/*
 * 多行注释
 * 常用于临时屏蔽大段代码
 */
int main() {
    std::cout << "Hello" << std::endl;
    return 0;
}
```
