# 1.4 控制流

## 本节讲什么

本节讲解「控制流」。

## 要点

- `while` 循环
- `for` 循环
- `if` 条件语句
- 利用循环读取数量不定的输入数据

## 示例

```cpp
#include <iostream>
int main() {
    int sum = 0, val = 1;
    while (val <= 10) {
        sum += val;
        ++val;
    }
    std::cout << "Sum 1..10 = " << sum << std::endl;
    return 0;
}
```
