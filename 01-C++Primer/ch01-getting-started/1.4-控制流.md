# 1.4 控制流

## 本节讲什么

介绍允许程序跳出顺序执行、实现更复杂执行路径的语句。详细讲解 `while` 循环（反复检测条件并执行循环体直至条件为假）、`for` 循环（将初始化、循环条件和变量修改集中在循环头以简化代码）以及支持条件执行的 `if` 语句。还演示了如何使用 `while (std::cin >> value)` 不断读取数量不定的输入，直至遇到文件末尾（EOF）。

## 要点

- `while`：先判断条件，为真则重复执行循环体
- `for`：循环头包含初始化、条件、迭代语句，适合计数循环
- `if` / `else`：按条件选择执行路径
- `while (std::cin >> value)`：循环读取输入，直至 EOF 或输入失败
- Windows 下 EOF 常用 `Ctrl+Z` 后回车；Unix 下为 `Ctrl+D`

## 示例

```cpp
#include <iostream>
int main() {
    // while：累加 1 到 10
    int sum = 0, val = 1;
    while (val <= 10) {
        sum += val;
        ++val;
    }
    std::cout << "Sum 1..10 = " << sum << std::endl;

    // for：同样效果
    sum = 0;
    for (int i = 1; i <= 10; ++i)
        sum += i;
    std::cout << "Sum 1..10 = " << sum << std::endl;

    // if：按分数输出等级
    int score = 85;
    if (score >= 90)
        std::cout << "A" << std::endl;
    else if (score >= 60)
        std::cout << "Pass" << std::endl;
    else
        std::cout << "Fail" << std::endl;

    return 0;
}
```
