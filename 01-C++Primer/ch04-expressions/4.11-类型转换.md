# 4.11 类型转换

## 本节讲什么

本节讲解「类型转换」。

## 要点

- **隐式类型转换**：运算对象类型不一致时编译器自动转换（算术转换、数组转指针、指针转换、转为布尔类型等）
- **显式类型转换**：`static_cast`、`const_cast` 等

## 示例

```cpp
#include <iostream>
int main() {
    double d = 3.14;
    int i = static_cast<int>(d);  // 显式类型转换
    std::cout << i << std::endl;
    return 0;
}
```
