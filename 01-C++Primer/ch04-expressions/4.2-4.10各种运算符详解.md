# 4.2–4.10 各种运算符详解

## 本节讲什么

本节讲解「–4.10 各种运算符详解」。

## 要点

| 类别 | 内容 |
|------|------|
| 算术运算符 | `+`、`-`、`*`、`/`、`%` |
| 逻辑和关系运算符 | `&&`、`||`、`<`、`==` 等；逻辑与/或的**短路求值** |
| 赋值运算符 | `=`、`+=` 等复合形式 |
| 递增和递减 | `++`、`--`；前置与后置区别；建议优先使用前置版本 |
| 成员访问 | 点运算符 `.`、箭头运算符 `->` |
| 条件运算符 | `?:` 及嵌套用法 |
| 位运算符 | `~`、`<<`、`>>`、`&`、`^`、`|` |
| sizeof | 返回表达式或类型所占字节数 |
| 逗号运算符 | `,` |

## 示例

```cpp
#include <iostream>
int main() {
    int i = 10;
    i += 2;   // 复合赋值
    ++i;      // 前置递增
    int j = i++;  // 后置递增
    std::cout << i << " " << j << std::endl;
    return 0;
}
```
