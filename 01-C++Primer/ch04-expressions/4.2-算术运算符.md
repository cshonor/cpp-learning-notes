# 4.2 算术运算符

## 本节讲什么

`+` `-` `*` `/` `%` 及一元 `+` `-` 的规则与易错点。

## `+` `-` `*` `/`

- **整数除法**截断小数：`5 / 2 == 2`
- **浮点除法**保留小数：`5.0 / 2 == 2.5`
- 混合类型会**隐式转换**（见 [4.11.1](./4.11.1-隐式类型转换.md)）
- 一元 `-x`：取相反数
- 一元 `+x`：几乎无实际作用（可触发整型提升）

## `%` 取余

- **仅整型**；浮点不能用 `%`
- 余数符号与**被除数**一致：`-5 % 2 == -1`

## 易错点

| 坑 | 说明 |
|----|------|
| **除 0** | 未定义行为 |
| 负数取余 | 符号跟被除数，不是数学模运算 |
| 混合类型 | 自动提升，可能意外变 `double` |

**HFT**：价格比例用整数除法前先确认是否该用浮点；用 `if (b != 0 && a / b)` 防除零（见 [4.3](./4.3-关系与逻辑运算符.md)）。
