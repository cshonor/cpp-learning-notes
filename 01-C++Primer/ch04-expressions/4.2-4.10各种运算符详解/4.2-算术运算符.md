# 4.2 算术运算符

## 本节讲什么

`+` `-` `*` `/` `%` 及一元 `+` `-` 的规则与易错点。

## `+` `-` `*` `/`

- **整数除法**截断小数：`5 / 2 == 2`
- **浮点除法**保留小数：`5.0 / 2 == 2.5`
- 混合类型会**隐式转换**（见 [4.11.1](../4.11-类型转换/4.11.1-隐式类型转换.md)）
- 一元 `-x`：取相反数
- 一元 `+x`：几乎无实际作用（可触发整型提升）

## `%` 取余

- **仅整型**；浮点不能用 `%`
- 余数符号与**被除数**一致：`-5 % 2 == -1`

## 易错点

| 坑 | 说明 |
|----|------|
| **除 0** | 未定义行为 |
| 负数取余 | 符号跟被除数，不是数学模运算 |
| 混合类型 | 自动提升，可能意外变 `double` |

**HFT**：价格比例用整数除法前先确认是否该用浮点；用 `if (b != 0 && a / b)` 防除零（见 [4.3](./4.3-关系与逻辑运算符.md)）。

## 示例

### 整数除法 vs 浮点除法

```cpp
#include <iostream>

int main() {
    std::cout << 5 / 2 << "\n";      // 2  整数除法，截断小数
    std::cout << 5.0 / 2 << "\n";    // 2.5
    std::cout << 5 / 2.0 << "\n";    // 2.5  int 提升为 double

    double price = 3750.5;
    int lots = 10;
    // std::cout << price / lots;    // 375.05 浮点结果
    std::cout << static_cast<int>(price) / lots << "\n";  // 375 先截断再除
    return 0;
}
```

### 取余 `%`

```cpp
#include <iostream>

int main() {
    std::cout << 10 % 3 << "\n";    // 1
    std::cout << -10 % 3 << "\n";   // -2  符号跟被除数
    std::cout << 10 % -3 << "\n";   // 1

    int total_qty = 25;
    int lot_size = 10;
    int full_lots = total_qty / lot_size;   // 2 整手
    int remainder = total_qty % lot_size;   // 5 零股
    std::cout << full_lots << " " << remainder << "\n";
    return 0;
}
```

### 一元 `+` `-` 与混合运算

```cpp
#include <iostream>

int main() {
    int x = 5;
    std::cout << -x << "\n";         // -5
    std::cout << +x << "\n";         // 5（一元 +，几乎无效果）

    char c = 'A';
    auto n = c + 1;                  // char 提升为 int → 66
    std::cout << n << "\n";

    return 0;
}
```

### 防除零（配合逻辑短路）

```cpp
int a = 100, b = 0;
if (b != 0 && a / b > 10) {   // b==0 时不会算 a/b
    /* ... */
}
// int bad = a / b;             // ❌ b==0 → UB
```

### 与优先级结合（见 [4.12.1](../4.12-运算符优先级表/4.12.1-优先级与结合律表.md)）

```cpp
int a = 2, b = 3;
std::cout << a + b * 4 << "\n";   // 14，先算 b*4 再加 a
std::cout << (a + b) * 4 << "\n"; // 20，括号改变结合
```
